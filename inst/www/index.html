metabox: A toolbox for analysis, interpretation and integrative exploration of omic data

An R-based web applications for data processing, statistical analysis, integrative visual exploration and functional analysis with several approaches (such as set enrichment analysis, overrepresentation analysis and WordCloud generation).

Database location

Your current database is set to the following location:


                
Use the following steps to set a database location:
                
    
                  
  • Create a file database.R in the folder HOME/USER_NAME/SUBFOLDER_NAME/ 
    
                  e.g. "Users/myname/Documents/database.R"
    • 
                  
  • Copy and paste the following line to the database.R 
    
                  database.location = "URL OF YOUR DATABASE LOCATION" 
    
                  e.g. database.location = "http://mylocation.com:7474/db/data/"
    • 
                  
  • Reload the web browser
    •
